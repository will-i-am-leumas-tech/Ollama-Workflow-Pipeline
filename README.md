# Ollama Pipeline: HUD & SDK

A powerful, visual, and programmatic automation engine for Ollama. Create complex multi-step AI pipelines with loops, JSON parsing, and structured file outputs.

## 🚀 Three Ways to Play

1.  **Web HUD (The Builder)**: A beautiful, dark-themed web interface to build, run, and manage your workflows visually.
2.  **Node.js SDK**: Integrate the pipeline engine directly into your own applications with a simple NPM-like package.
3.  **Interactive CLI**: Run workflows from your terminal with a step-by-step interactive experience.

---

## 🎨 Web HUD (Heads-Up Display)
The HUD is a full-featured management suite for your AI pipelines.

- **Visual Builder**: Drag-and-drop logic for adding Prompts and Loops.
- **AI Scaffolding**: Use the "Auto-Generate" feature to have Ollama design a workflow for you from a simple prompt.
- **Run & Log**: Execute pipelines in the browser with a real-time console log and final state inspector.
- **Output Gallery**: Browse every file generated by your workflows, complete with metadata and live previews.
- **Favorites**: Star your most-used pipelines to keep them at the top of your dashboard.

**To Start the HUD:**
```bash
npm run server
```
Visit `http://localhost:3000` to get started.

---

## 📦 Node.js SDK
You can now use `ollama-pipeline` as a library in any other Node.js project.

```javascript
import { OllamaPipeline } from 'leumas-ollama-workflow-engine';

const result = await OllamaPipeline.run('creative/album-generator', {
  answers: { artistName: 'Neon Dreams', songCount: 5 },
  onLog: (type, msg) => console.log(`[${type}] ${msg}`)
});

console.log('Final State:', result.state);
```

**SDK Features:**
- **Simple API**: High-level `run()` method for one-shot execution.
- **Customizable**: Override models, output directories, and workflow locations.
- **Logging Hooks**: Capture pipeline progress in your own app's logging system.

See [docs/SDK_GUIDE.md](docs/SDK_GUIDE.md) for more.

## 🛠 Workflow Architecture
Define complex logic in simple JSON. 

- **Internal State**: Each step saves its result to a shared context.
- **JSON Parsing**: Automatically turn LLM responses into objects for looping.
- **Smart Templates**: Use `{{variables}}` and helpers like `{{slugify:name}}`.
- **Loops**: Iterate over arrays to generate multiple files in one run.

See [docs/WORKFLOW_GUIDE.md](docs/WORKFLOW_GUIDE.md) for the full schema.

## 🚦 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [Ollama](https://ollama.com/) running locally.

### Setup
1. Clone the repo and `npm install`.
2. Start the HUD: `npm run server`.
3. Build your first pipeline at `http://localhost:3000/builder.html`.

### Run via CLI
```bash
node runWorkflow.js creative/album-generator --set artistName="Synth Wave"
```

## 📜 Folder Structure
- `workflows/`: Your pipeline library (.json).
- `outputs/`: Generated files and run metadata.
- `server/`: HUD backend and frontend.
- `src/`: Core engine, SDK, and adapters.
- `docs/`: In-depth guides.
