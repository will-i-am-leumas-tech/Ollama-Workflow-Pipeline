# Custom Tool Integration Guide

The `ollama-pipeline` supports advanced multi-modal tools (OCR, Image Generation, 3D, etc.) through a plug-and-play Tool Registry.

## 1. Creating a New Tool
To add a new tool, create a `.js` file in the `src/tools/` directory.

### Tool Template
Every tool must export an asynchronous `run` function.

```javascript
/**
 * @param {Object} params - Rendered parameters from the workflow JSON
 * @param {Object} context - The current internal state of the pipeline
 * @returns {Promise<any>} - The result (object, string, or buffer)
 */
export const run = async (params, context) => {
  // 1. Your logic here (e.g., calling a local SDK)
  const result = await myLocalSdk.generate(params.prompt);

  // 2. Return the data
  return result;
};
```

## 2. Using the Tool in a Workflow
In your workflow JSON, add a step with `type: "tool"`.

```json
{
  "id": "my_tool_step",
  "type": "tool",
  "tool": "dummy", 
  "params": {
    "message": "Hello from {{albumTitle}}!",
    "nested": {
        "val": "{{someVariable}}"
    }
  },
  "output": {
    "directory": "./outputs/custom",
    "fileNameTemplate": "tool-result",
    "extension": ".json"
  }
}
```

### Key Properties:
- **`tool`**: (Required) The filename of your tool in `src/tools/` (without `.js`).
- **`params`**: (Optional) Data passed to the tool. All strings in this object are automatically rendered as templates using the current pipeline state.
- **`id`**: (Optional) The tool's return value will be saved to the pipeline state with this key.
- **`output`**: (Optional) If provided, the engine will automatically save the tool's result to a file.

## 3. Passing Data Between LLMs and Tools
You can chain tools and LLMs seamlessly:
1. **Ollama -> Tool**: Pass a description generated by Ollama into an image generator tool.
2. **Tool -> Ollama**: Pass the output of an OCR tool into an Ollama prompt for summarization.

## 4. Error Handling
If a tool throws an error, the engine will catch it, log it, and (if an `id` is provided) save the error message to the state so subsequent steps can handle it.
